# 📸 SnapScape - Premium Image Gallery

## 🌟 Overview

SnapScape is a modern and responsive image gallery built using HTML, CSS, and JavaScript. It provides a clean user interface with image categories, search, lightbox view, smooth animations, and responsive design.

---

## 🚀 Features

- Responsive Image Gallery
- Image Categories & Filters
- Live Search
- Lightbox Image Viewer
- Previous / Next Navigation
- Hover Effects
- Smooth Animations
- Modern UI Design
- Mobile Friendly

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
